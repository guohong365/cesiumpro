TransformHelper

new CesiumPro.TransformHelper(options)
core/TransformHelper.js 142

模型编辑控制器，虽然它也适用于点元素
Name Type Description
options object optional 具有以下属性
Name Type Default Description
lineWidth number 15 optional 线宽
xAxisColor Cesium.Color Cesium.Color.RED optional X轴的颜色
yAxisColor Cesium.Color Cesium.Color.GREEN optional Y轴颜色
zAxisColor Cesium.Color Cesium.Color.BLUE optional Z轴颜色
scaleAxisColor Cesium.Color Cesium.Color.WHITE optional 缩放轴的颜色
activeAxisColor Cesium.Color Cesium.Color.YELLOW optional 被激活的坐标标轴颜色
translateEnabled boolean true optional 是否创建平移控制器
rotateEnabled boolean false optional 是否创建旋转控制器，仅对模型可用
scaleEnabled boolean false optional 是否创建缩放控制器，仅对模型有效
xAxisLength number optional X轴的长度, 如果未定义将自动计算
yAxisLength number optional Y轴的长度，如果未定义将自动计算
zAxisLength number optional Z轴长度，如果未定义将自动计算
scaleAxisLength number optional 缩放轴的长度，如果未定义将自动计算
rotatePlaneRadius number optional 旋转轴的半径，如果未定义将自动计算
sizeInPixel boolean false optional 控制器的大小是否以像素为单位
originOffset Cesium.Cartesian3 Cesium.Cartesian3.ZERO optional 默认原点在点或模型的中心位置，该值设置相对于默认位置的偏移
Example:
const helper = new CesiumPro.TransformHelper({            
    rotateEnabled: true,
    translateEnabled: true,
    scaleEnabled: true,
    xAxisLength: 10,
    yAxisLength: 10,
    zAxisLength: 10,
    scaleAxisLength: 10,
    rotatePlaneRadius: 8
});

helper.addTo(viewer);
helper.bind(model);
