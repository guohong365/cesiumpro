WMSLayer

new CesiumPro.WMSLayer(options)
layer/WMSLayer.js 36

创建一个从WMS服务请求数据的图层。
Name Type Description
options * 具有以下属性
Name Type Default Description
url Resource | String 一个WMS服务的URL. 这个url支持和 XYZLayer一样的关键字。
layers String 要包含的图层，以逗号分割。
parameters Object WMSLayer.DefaultParameters optional GetMap URL中包含的额外参数。
getFeatureInfoParameters Object WMSLayer.GetFeatureInfoDefaultParameters optional GetFeatureInfo URL中包含的额外参数。
enablePickFeatures Boolean true optional 图层是否允许被pick WMSLayer#pickFeatures
getFeatureInfoFormats Array.<GetFeatureInfoFormat> WMSLayer.DefaultGetFeatureInfoFormats optional WMS GetFeatureInfo 请求的格式。
rectangle Rectangle Cesium.Rectangle.MAX_VALUE optional 图层的范围。
tilingScheme TilingScheme new Cesium.GeographicTilingScheme() optional 图层坐标系
tileWidth Number 256 optional 瓦片宽度
tileHeight Number 256 optional 瓦片高度
minimumLevel Number 0 optional 最小层级
maximumLevel Number optional 最大层级
crs String optional CRS 规范，用于 WMS version >= 1.3.0。
srs String optional SRS 规范，用于 WMS version 1.1.0 或 1.1.1
credit Credit | String optional A credit for the data source, which is displayed on the canvas.
subdomains String | Array.<String> 'abc' optional {s}占位符可用的子域名，如果该值为字符串，由字符串中的每个字符都是一个子域名。
clock Clock optional 在确定时间维度的值时使用的 Clock 实例。指定 `times` 时需要。
times TimeIntervalCollection optional TimeIntervalCollection 及其数据属性是一个包含时间动态维度及其值的对象。
Example:
var provider = new Cesium.WebMapServiceImageryProvider({
  url : 'https://sampleserver1.arcgisonline.com/ArcGIS/services/Specialty/ESRI_StatesCitiesRivers_USA/MapServer/WMSServer',
  layers : '0',
  proxy: new Cesium.DefaultProxy('/proxy/')
});

viewer.addLayer(provider);
