pickPosition

pickPosition(pixel, viewer) → Cesium.Cartesian3
pickPosition.js 13

坐标拾取函数,如果pixel所在位置有Primitive或Entity，将获取Primitive或Entity上的位置，否则获取球面坐标。 简言之,如果点击在模型上将获得模型上的坐标，否则获取球面坐标。

Note:获取模型上的坐标时需要打开地形深度调整，否则获取的点位不准。

Name Type Description
pixel Cesium.Cartesian3 屏幕坐标
viewer Cesium.Viewer Viewer对象
Returns:
笛卡尔坐标
