new CesiumPro.MultipleTerrainProvider(terrainList, options)
core/MultipleTerrainProvider.js 82

为场景添加多个地形数据。 todo: 目前两个地形相接的的一行瓦片数据丢失。
Name Type Description
terrainList MultipleTerrainProvider.TerrainList 地形列表
options Object 具有以下属性
Name Type Default Description
requestVertexNormals Boolean false optional 是否请求法线数据
requestWaterMask Boolean false optional 是否请求水面数据
requestMetadata Boolean true optional 是否请求元数据
ellipsoid Cesium.Ellipsoid optional 椭球体
credit String | Cesium.Credit optional 信用信息
Example:
const terrain = new CesiumPro.MultipleTerrainProvider([
        {
            url: '../data/terrain/terrain-left',
            zIndex:3
        },
        {
            url: '../data/terrain/terrain-right',
            zIndex:2
        }
    ])
terrain.readyPromise.then(() => {
    terrain.createBoundingRectangle(viewer);
})
viewer.terrain = terrain;
