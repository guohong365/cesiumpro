RectangularSensorPrimitive

new CesiumPro.RectangularSensorPrimitive(options)
scene/RectangularSensorPrimitive.js 78

模拟相控阵雷达。
Name Type Description
options Object 具有以下属性
Name Type Default Description
show Boolean optional 是否显示
position Cartesian3 | LonLat optional 图形位置
modelMatrix Matrix4 optional 模型矩阵,如果定义，则覆盖position属性
slice Number 32 optional 切分程度
radius Number 1 optional 半径
lineColor Color Color.RED optional 线的颜色
xHalfAngle Number optional 水平半夹角,单位度
xyHalfAngle Number optional 垂直半夹角，单位度
showSectorLines Boolean true optional 是否显示扇面线
showSectorSegmentLines Boolean true optional 是否显示扇面和圆顶面连接线
showLateralSurfaces Boolean true optional 是否显示侧面
material Material Material.ColorType optional 材质
lateralSurfaceMaterial Material Material.ColorType optional 侧面材质
showDomeSurfaces Boolean true optional 是否显示圆弧顶表面
domeSurfaceMaterial Material Material.ColorType optional 圆弧顶表面材质
showDomeLines Boolean true optional 是否显示圆弧顶表面线
showIntersection Boolean false optional 是否显示与地球相交的线
intersectionColor Color optional 与地球相交的线的颜色
intersectionWidth Number 5 optional 与地球相交的线的宽度
showThroughEllipsoid Boolean false optional 是否穿过地球
showScanPlane Boolean true optional 是否显示扫描面
scanPlaneColor Color Color.WHITE optional 扫描面颜色
scanPlaneMode String 'H' optional 扫描方向，H表示水平扫描，V表示垂直扫描
speed Number 10 optional 扫描速度，值越大，扫描越快
