new CesiumPro.WindField(viewer, options)
scene/windField/WindField.js 41

3D 风场可视化
Name Type Description
viewer Cesium.Viewer
options object 具有以下属性
Name Type Default Description
color Cesium.Color Cesium.Color.WHITE optional 粒子颜色
maxParticles number 128 * 128 optional 最大粒子数
particleHeight number 100 optional 粒子高度
fadeOpacity number 0.996 optional 渐变不透明度
dropRate number 0.003 optional
dropRateBump number 0.01 optional
speedFactor number 4.0 optional
lineWidth number 4.0 optional 线宽
color object optional 颜色表
particlesTextureSize number 128 optional 粒子纹理大小,必须为2^n
Example:
const wind = new CesiumPro.WindField({
  maxParticles: 128 * 128,
  particleHeight: 100.0,
  fadeOpacity: 0.996,
  dropRate: 0.003,
  dropRateBump: 0.01,
  speedFactor: 4.0,
  lineWidth: 4.0
})
