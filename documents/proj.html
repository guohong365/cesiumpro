static CesiumPro.proj.get(epsgCode, options) → Cesium.TilingScheme
core/proj.js 90

通过epsg code获得Cesium预设的坐标系统
Name Type Default Description
epsgCode string 'EPSG:3857' optional epsg码，EPSG:3857获得Web墨卡托投影，EPSG:4326获得WGS84坐标系
options * optional 定义坐标系的详细参数，具体请参考WebMercatorTilingScheme和GeographicTilingScheme
Returns:
相应epsg code的坐标系
Example:
new XYZLayer({
      url: Url.buildModuleUrl('assets/tiles/{z}/{x}/{y}.png'),
      maximumLevel: 4,
      tilingScheme: CesiumPro.proj.get("EPSG：3857")
 })
}
