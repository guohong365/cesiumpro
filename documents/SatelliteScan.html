Name Type Default Description
show Bool true optional 是否可见
position Cartesian3 optional 卫星的位置
path Entity optional 卫星运动轨道,该属性将覆盖position
radius Number optional 卫星在地面的扫描半径
inverse Bool false optional 扫描方向，该值为true卫星将从下向上扫描
tracked Bool false optional 是否跟踪卫星
color Color optional 扫描波颜色
option.image Bool optional 卫星图标，如果定义了model该属性将不生效
options.model Bool false optional 卫星模型
option.axis Cartesian3 Cartesian3(1,0,0) optional 卫星扫描方向偏转的坐标轴
options.rotation Number optional 扫描波在方向偏转角度
option.animate Bool true optional 是否动态扫描
options.thickness number 0.3 optional 扫描波每一片的厚度
options.slices number 20 optional 扫描波的分片数量
