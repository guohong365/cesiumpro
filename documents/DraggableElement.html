DraggableElement

new CesiumPro.DraggableElement(container, target)
core/DraggableElement.js 12

创建一个可拖拽的DOM元素,该元素的position必须为absolute或fixed;
Name Type Default Description
container String 要移到的元素的选择器
target String container optional 监听鼠标事件的元素，一般是标题栏
Example:
new DraggableElement('#tool-panel')
new DraggableElement('#tool-panel','.tool-header-class')
