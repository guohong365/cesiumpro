XYZLayer

new CesiumPro.XYZLayer(options)
layer/XYZLayer.js 129

创建一个image图层，该图层通过使用X/Y/Z指定的URL模板请求图像。你可以用它来请求TMS,WMS标准的地图服务，甚至WMTS。
Name Type Description
options Promise.<Object> | Object 具有以下属性
Name Type Default Description
url Resource | String 瓦片的url模板，它支持以下关键字:
  • {z}: 瓦片的层级，第一层为0级
  • {x}: 切片方案中X的坐标，最左边为0
  • {y}: 切片方案中Y的坐标，最上边为0
  • {s}: 子域名
  • {reverseX}: 切片方案中X的坐标，最右边为0
  • {reverseY}: 切片方案中Y的坐标，最下边为0
  • {reverseZ}: 瓦片的层级，最大层级为0
  • {westDegrees}: 瓦片最左边的坐标，单位度
  • {southDegrees}: 瓦片最下边的坐标，单位度
  • {eastDegrees}: 瓦片最右边的坐标，单位度
  • {northDegrees}: 瓦片最上边的坐标，单位度
  • {westProjected}: 瓦片最左边的坐标，单位米
  • {southProjected}: 瓦片最下边的坐标，单位米
  • {eastProjected}: 瓦片最右边的坐标，单位米
  • {northProjected}: 瓦片最上边的坐标，单位米
  • {width}: 瓦片宽度，单位像素
  • {height}: 瓦片高度，单位像素
pickFeaturesUrl Resource | String optional 用于选择功能的 URL 模板，如果未定义 XYZLayer#pickFeatures 将返回undefined，表示未选中任何内容，该模板除了支持url的所有关键字外，还支持以下关键字:
  • {i}: 选择位置的列，单位像素，最左边为0。
  • {j}: 拾取位置的行，单位像素，最上边为0。
  • {reverseI}: 选择位置的列，单位像素，最右边为0。
  • {reverseJ}: 拾取位置的行，单位像素，最下边为0
  • {longitudeDegrees}: 拾取位置的经度，单位度
  • {latitudeDegrees}: 拾取位置的纬度，单位度
  • {longitudeProjected}: 拾取位置的经度，单位米
  • {latitudeProjected}: 拾取位置的纬度，单位米
  • {format}: 一个函数，用于定义信息的返回形式, 如Cesium.GetFeatureInfoFormat.
urlSchemeZeroPadding Object optional 设置坐标的位数，不足的以0填充，对于urlSchemeZeroPadding : { '{x}' : '0000'} 如果x为12，将被填充为0012。它具有以下关键字：
  • {z}: z关键字的填充方案
  • {x}: x关键字的填充方案
  • {y}: y关键字的填充方案
  • {reverseX}: reverseX关键字的填充方案
  • {reverseY}: reverseY关键字的填充方案
  • {reverseZ}: reverseZ关键字的填充方案
subdomains String | Array.<String> 'abc' optional {s}占位符可用的子域名，如果该值为字符串，由字符串中的每个字符都是一个子域名。
credit Credit | String '' optional 数据源的版权信息.
minimumLevel Number 0 optional 图层支持的最低细节级别。在指定最小级别的瓦片数量很少时要小心，较大的数字可能会导致渲染问题。
maximumLevel Number optional 图层支持的最大细节级别，如果未定义，则无限制。
rectangle Rectangle Cesium.Rectangle.MAX_VALUE optional 瓦片覆盖范围，单位弧度。
tilingScheme TilingScheme Cesium.WebMercatorTilingScheme optional 图层的坐标系，默认为墨卡托投影。
tileWidth Number 256 optional 瓦片宽度。
tileHeight Number 256 optional 瓦片高度。
hasAlphaChannel Boolean true optional 瓦片图像是否拥有alpha通道，如果为false，内存使用量和加载时间将减少，但是会失去alpha通道的数据。
getFeatureInfoFormats Array.<GetFeatureInfoFormat> XYZLayer.defaultFeatureInfoFormats optional XYZLayer#pickFeatures 调用时在指定位置获取特征信息的格式。如果未指定此参数，则禁用特征拾取。
enablePickFeatures Boolean true optional 是否支持要素不支持，如果为true,XYZLayer#pickFeatures将请求pickFeaturesUrl并解析其内容，否则将返回undefined。
customTags Object optional 为URL模板自定义关键字。该对象的键必须为字符串，值为函数。
Example:
// Access Natural Earth II imagery, which uses a TMS tiling scheme and Geographic (EPSG:4326) project
const tms = new CesiumPro.XYZLayer({
    url : Cesium.buildModuleUrl('Assets/Textures/NaturalEarthII') + '/{z}/{x}/{reverseY}.jpg',
    credit : '© Analytical Graphics, Inc.',
    tilingScheme : new Cesium.GeographicTilingScheme(),
    maximumLevel : 5
});
// Access the CartoDB Positron basemap, which uses an OpenStreetMap-like tiling scheme.
const positron = new CesiumPro.XYZLayer({
    url : 'http://{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}.png',
    credit : 'Map tiles by CartoDB, under CC BY 3.0. Data by OpenStreetMap, under ODbL.'
});
// Access a Web Map Service (WMS) server.
const wms = new CesiumPro.XYZLayer({
   url : 'https://programs.communications.gov.au/geoserver/ows?tiled=true&' +
         'transparent=true&format=image%2Fpng&exceptions=application%2Fvnd.ogc.se_xml&' +
         'styles=&service=WMS&version=1.1.1&request=GetMap&' +
         'layers=public%3AMyBroadband_Availability&srs=EPSG%3A3857&' +
         'bbox={westProjected}%2C{southProjected}%2C{eastProjected}%2C{northProjected}&' +
         'width=256&height=256',
   rectangle : Cesium.Rectangle.fromDegrees(96.799393, -43.598214999057824, 153.63925700000001, -9.2159219997013)
});
// Using custom tags in your template url.
const custom = new CesiumPro.XYZLayer({
   url : 'https://yoururl/{Time}/{z}/{y}/{x}.png',
   customTags : {
       Time: function(imageryProvider, x, y, level) {
           return '20171231'
       }
   }
});
// geoserver tms服务
const  geoserverTMS = new XYZLayer({
     url: 'http://localhost:8080/geoserver/gwc/service/tms/1.0.0/topp%3Astates@EPSG%3A4326@png/{z}/{x}/{reverseY}.png',
     tilingScheme: proj.get('EPSG:4326')
 })
// geoserver wms服务，并支持pick
const geoserverTMS = new XYZLayer({
     pickFeaturesUrl:'http://localhost:8080/geoserver/wms?service=WMS&version=1.1.1&request=GetFeatureInfo&layers=topp%3Astates&bbox={westProjected}%2C{southProjected}%2C{eastProjected}%2C{northProjected}&width={width}&height={height}&srs=EPSG%3A4326&query_layers=topp%3Astates&info_format={format}&x={i}&y={j}',
     url: 'http://localhost:8080/geoserver/topp/wms?service=WMS&version=1.1.1&request=GetMap&layers' +
         '=topp%3Astates&bbox={westDegrees}%2C{southDegrees}%2C{eastDegrees}%2C{northDegrees}&transparent=true&' +
          'width=768&height=330&srs=EPSG%3A4326&format=image/png',
     tilingScheme: proj.get('EPSG:4326'),
     getFeatureInfoFormats: [new Cesium.GetFeatureInfoFormat("xml", "text/xml")]
})
// geoserver WMTS服务
const layer = new XYZLayer({
    url: 'http://localhost:8080/geoserver/gwc/service/wmts?layer=topp%3Astates&style=&til' +
        'ematrixset=EPSG%3A4326&Service=WMTS&Request=GetTile&Version=1.0.0&Format=image%2Fpng&TileMatrix={TileMatrix}' +
        '&TileCol={tileCol}&TileRow={tileRow}',
    tilingScheme: proj.get('EPSG:4326'),
    customTags: {
        TileMatrix: function (imageryProvider, x, y, level) {
            return 'EPSG:4326:' + level
        },
        tileCol: function (imageryProvider, x, y, level) {
            return y
        },
        tileRow: function (imageryProvider, x, y, level) {
            return x
        }
    }
})
