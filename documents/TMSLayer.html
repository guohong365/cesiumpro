TMSLayer

new CesiumPro.TMSLayer(options)
layer/TMSLayer.js 42

创建一个从TMS服务请求数据的图层。
Name Type Description
options Promise.<Object> | Object 具有以下属性
Name Type Default Description
url Resource | String TMS服务地址或，瓦片的url模板，如果是url模板，它支持以下关键字:
  • {z}: 瓦片的层级，第一层为0级
  • {x}: 切片方案中X的坐标，最左边为0
  • {y}: 切片方案中Y的坐标，最上边为0
  • {s}: 子域名
  • {reverseX}: 切片方案中X的坐标，最右边为0
  • {reverseY}: 切片方案中Y的坐标，最下边为0
  • {reverseZ}: 瓦片的层级，最大层级为0
  • {westDegrees}: 瓦片最左边的坐标，单位度
  • {southDegrees}: 瓦片最下边的坐标，单位度
  • {eastDegrees}: 瓦片最右边的坐标，单位度
  • {northDegrees}: 瓦片最上边的坐标，单位度
  • {westProjected}: 瓦片最左边的坐标，单位米
  • {southProjected}: 瓦片最下边的坐标，单位米
  • {eastProjected}: 瓦片最右边的坐标，单位米
  • {northProjected}: 瓦片最上边的坐标，单位米
  • {width}: 瓦片宽度，单位像素
  • {height}: 瓦片高度，单位像素
fileExtension String 'png' optional 图像的扩展名
credit Credit | String '' optional 数据源的版权信息.
minimumLevel Number 0 optional 图层支持的最低细节级别。在指定最小级别的瓦片数量很少时要小心，较大的数字可能会导致渲染问题。
maximumLevel Number optional 图层支持的最大细节级别，如果未定义，则无限制。
rectangle Rectangle Cesium.Rectangle.MAX_VALUE optional 瓦片覆盖范围，单位弧度。
tilingScheme TilingScheme Cesium.WebMercatorTilingScheme optional 图层的坐标系，默认为墨卡托投影。
tileWidth Number 256 optional 瓦片宽度。
tileHeight Number 256 optional 瓦片高度。
flipXY Boolean optional 是否翻转了XY轴
Example:
const layer = new TMSLayer({
  url: 'http://localhost:8080/geoserver/gwc/service/tms/1.0.0/topp%3Astates@EPSG%3A4326@png',
  tilingScheme: proj.get('EPSG:4326')
}
