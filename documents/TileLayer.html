TileLayer

new CesiumPro.TileLayer(options)
layer/TileLayer.js 28

创建一个从WMS服务请求数据的图层。
Name Type Description
options * 具有以下属性
Name Type Default Description
enablePickFeatures Boolean true optional 图层是否允许被pick TileLayer#pickFeatures
rectangle Rectangle Cesium.Rectangle.MAX_VALUE optional 图层的范围。
tileWidth Number 256 optional 瓦片宽度
tileHeight Number 256 optional 瓦片高度
minimumLevel Number 0 optional 最小层级
maximumLevel Number optional 最大层级
Example:
var provider = new Cesium.WebMapServiceImageryProvider({
  url : 'https://sampleserver1.arcgisonline.com/ArcGIS/services/Specialty/ESRI_StatesCitiesRivers_USA/MapServer/WMSServer',
  layers : '0',
  proxy: new Cesium.DefaultProxy('/proxy/')
});

viewer.addLayer(provider);
